Calculatrice en ligne

Une calculatrice interactive développée en HTML, CSS et JavaScript.

 Fonctionnalités

 Addition
 Soustraction
 Multiplication
 Division
 Bouton effacer (C)
 Bouton supprimer (⌫)
Interface moderne et intuitive
 Design responsive

 Technologies utilisées

- HTML5
- CSS3 (Grid & Flexbox)
- JavaScript (DOM manipulation)

 Objectif du projet

Ce projet a été réalisé dans le cadre de mon apprentissage en développement web afin de pratiquer :
- La manipulation du DOM
- La gestion des événements
- La structuration d’un projet front-end
- Le design d’interface utilisateur

 Projet réalisé pour progresser en développement front-end.
